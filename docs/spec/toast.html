Component · Toast

Toast.

The transient acknowledgment. Floating surface — symmetric 16px corners, soft amethyst-tinted drop shadow — sibling of Modal in the surface family. Status comes from color alone, sourced from the shared semantic palette. One at a time: queue, don't stack.

Motion · placement · timing

Placement

Bottom-center over the current screen. One at a time — if multiple messages arrive, queue them.

Timing

Default duration: 4000ms auto-dismiss. duration={0} pins until the user clicks dismiss — use for critical errors.

Enter

Slides up from bottom · duration-fast · ease-out-soft.

Exit

Fades to opacity 0 · duration-fast · onDismiss fires.

Accessibility
  • Container responsibility: role="status" for success/info, role="alert" for warning/error
  • Dismiss aria-label="Dismiss" always present · Enter and Space activate
  • 4000ms is enough for most readers — but escalate to duration={0} when the message must be acknowledged
  • All four type palettes pass WCAG AA on their bg
  • Don't put links, buttons, or form inputs inside the toast — message only. Actionable feedback belongs in inline UI or a Modal
