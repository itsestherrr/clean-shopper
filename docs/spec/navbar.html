Anatomy

Brand · Home · Browse

Plain wordmark (N-05). Page links are text only — no icons (N-01). No active indicator (N-02).

Global Search

Renders the shared SearchInput. URL-owned ?q= state (N-06). Pages read the param; they don't render their own input.

Favorites · Account

Icon-only Favorites + 36px amethyst-gradient avatar with initials. Click avatar → dropdown (N-04).

Search behavior — N-06

Off /search

Typing does not change the URL. Pressing Enter routes to /search?q=value.

On /search

Typing replaces ?q= live (replace: true) so results filter on every keystroke. Clearing the input zeroes ?q= without changing the route.

  • Focus rings system-wide: 2px amethyst outline, +2px offset, :focus-visible only
  • SearchInput's keyboard behavior is owned by SearchInput — Enter submits, × clears
