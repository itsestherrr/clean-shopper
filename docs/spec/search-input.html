Component · SearchInput

SearchInput.

The text input the user lives in. Pill-shaped, leading magnifier, trailing clear that only shows when there's something to clear. The system's single focus convention applies — 2px amethyst outline, 2px offset, :focus-visible only — no scale, no lift, no per-component flourish.

Accessibility
  • Native <input type="text"> — standard keyboard behavior, IME-friendly
  • Focus ring uses :focus-visible so a mouse click does not draw the ring; Tab navigation does
  • Clear control is an IconButton (ghost sm) — already carries aria-label="Clear search" and Enter/Space activation
  • Placeholder is never the sole label — page-level label sits above the input where applicable
  • Loading spinner is decorative; use page-level aria-live on the results region for the actual loading announcement
