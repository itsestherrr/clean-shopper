Component · EmptyState

EmptyState.

Surface for "no content yet" moments. Stroke icon, title, optional description, optional action — wrapped in a soft white card with a divider border. Empty isn't where the brand should sing, so this surface stays unsigned: no asym shadow, no lime, no amethyst disc. The frame's job is to anchor the message inside a page that already has chrome around it.

Accessibility
  • Pure presentational. Keyboard interaction only via the optional action button
  • Title renders as <h3> — parent page should not also place an h3 inside this surface
  • Icon is decorative — no aria-label; the title carries the meaning
  • Don't substitute emoji for an SVG glyph — emoji do not pick up theme color and read inconsistently to AT
  • Don't reuse this surface for loading or error — those states need their own pattern + role
