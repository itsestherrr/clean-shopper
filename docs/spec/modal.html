Behavior

Dismissal

Backdrop click · Escape key · ghost × in the header. All three call onClose.

Focus trap

First focusable element gets focus on open. Tab cycles inside the panel — Shift+Tab from the first element wraps to the last, and vice versa.

Body scroll lock

Sets document.body.style.overflow = 'hidden' while open and restores on close.

Accessibility
  • Panel has role="dialog" and aria-modal="true"
  • aria-label mirrors the title prop when present
  • Focus trap on open · first focusable element receives focus · Tab and Shift+Tab wrap inside the panel
  • Escape closes via document-level keydown listener; backdrop click closes via onClick on the overlay
  • Close button is an IconButton ghost md with aria-label="Close modal"
  • Body scroll is locked while open so background content doesn't drift behind the dialog
