Component · IngredientList · v4

IngredientList.

Single-column expandable rows. Status pinned in a fixed 100px column on the left so ingredient names line up vertically across the list. Click a row to reveal the safety reason. Personal-avoid items get an inline pill (In your avoid list) — distinct from the safety rating itself.

v4 — replaces both the old IngredientBar (deprecated) and the previous ingredient row pattern. The personal-avoid pill is inline markup pending a real PreferenceTag primitive.

Names align vertically because the status column is a fixed-width 100px element pinned to the left. Hover is light-touch — border darkens, bg shifts to cream. No transform, no shadow.

Hover signature scope DS-01

Lift + colored shadow is reserved for destination cards — ProductCard, hero CTAs. Things you click to go somewhere.

List rows like ingredients live inside a content surface and expand in place. They get a quiet hover only — border darken to amethyst-40, bg shift to cream. Anything stronger competes with the page.

Accessibility
  • Each row is a <button> inside a <li>. List wrapped in <ul role="list">
  • aria-expanded on the row reflects open state — chevron rotation is decorative
  • Tier signaled by dot color and the uppercase tier label text. Color is never the sole signal
  • Personal-avoid pill uses both avoid-tint bg and uppercase label text — readable to AT
  • Focus ring: amethyst, 2px, 2px offset
  • Loading skeletons are decorative — pair the parent region with aria-busy="true" while loading
